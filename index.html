Solo project: Sans's Nightmare Try it out!

This project was created while working in a project week on school. The requirements were finishing the project in the week time we were given. It had to be coded in a program called Greenfoot and published to the GreenFoot website.

GreenFoot is an integrated development environment using Java. In this week I learned to work better with Java to make an interactive game.

This game uses the w,a,s,d keys to walk into directions and I coded it in a way that when you press a movement key such as D the sprite would walk right. In this game I worked on the movements, movement animations, shooting, shooting animations, HP bar, score counter, heart tracking and music integration.
